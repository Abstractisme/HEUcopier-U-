# HEUcopier-U-
实现对U盘复制的一个小小脚本 
已实现： 
1.判断移动硬盘盘符，并且只复制移动硬盘数据 
2.自动上传至FTP服务器，无需另插u盘
3.静默运行 
4.无需管理员权限，适用于所有Windows系统。 
未实现：
1.判断文件大小，若大于某个值停止运行
2.延迟至设备插入后运行
3.绕过防火墙

本程序设计时适用于badusb场景，欢迎各位进行进一步完善。
如需使用，请下载release的压缩包，修改heu.bat中的数据，保存
再将解压并修改后的文件压缩，设置为自解压格式，设置解压后运行heu.vbs，设置静默运行。
即可得到成品
